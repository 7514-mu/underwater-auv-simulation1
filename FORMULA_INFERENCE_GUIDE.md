# 🌊 水温控制 - 公式推断实现详解

## 🎯 什么是"公式推断"

**定义**：不模拟真实的物理过程，用数学公式直接计算结果。

---

## 📐 具体实现流程

### **步骤1：温度输入**
```python
用户输入: 30°C
```

### **步骤2：公式计算（公式推断）**
```python
# 使用纯水密度公式
ρ = 1000 - 0.2 × (T - 4)²

代入 T = 30:
ρ = 1000 - 0.2 × (30 - 4)²
ρ = 1000 - 0.2 × 26²
ρ = 1000 - 0.2 × 676
ρ = 1000 - 135.2
ρ = 864.8

修正（考虑实际物理范围）：
ρ ≈ 996.2 kg/m³（查表/精确公式）
```

### **步骤3：应用到仿真**
```python
调用服务: /rexrov/set_fluid_density
参数: data = 996.2
```

### **步骤4：物理效果**
```python
仿真器内部:
F_浮力 = ρ × g × V
       = 996.2 × 9.8 × V
       < 998.2 × 9.8 × V (20°C时)

结果: 浮力减小 → 机器人下沉
```

## 🔍 对比：物理仿真 vs 公式推断

### **物理仿真**（如果做真实模拟）：
```
1. 建立水体3D网格（数百万个点）
2. 在每个网格求解热传导方程：
   ∂T/∂t = α∇²T
3. 在每个网格计算密度变化：
   ρ = f(T, P, S)
4. 在每个网格求解流体方程：
   ∂v/∂t = ...（Navier-Stokes）
5. 计算浮力、阻力等

计算时间：几小时到几天
需要的软件：OpenFOAM, ANSYS Fluent
```

### **公式推断**（当前实现）：
```
1. 输入温度 T = 30°C
2. 计算密度 ρ = f(T)（一个公式）
3. 应用到仿真

计算时间：< 0.001秒
需要的软件：Python + ROS 2
```

## 📊 使用的公式

### **温度 → 密度公式**
```python
# 纯水密度（简化版）
ρ = 1000 - 0.2 × (T - 4)²

# 纯水密度（精确版）
ρ = 999.842594 + 6.793952×10⁻²T
    - 9.095290×10⁻³T² + 1.001685×10⁻⁴T³
    - 1.120083×10⁻⁶T⁴ + 6.536332×10⁻⁹T⁵

# 海水密度（盐度35‰）
ρ = 淡水密度 + 4.788S - 1.117×10⁻²ST + 2.717×10⁻⁴ST²
```

### **密度 → 浮力公式**
```python
F_buoyancy = ρ × g × V

其中：
ρ = 水密度（变量）
g = 9.8 m/s²（重力加速度）
V = 排开水的体积（恒定）
```
