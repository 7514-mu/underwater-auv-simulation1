# 📡 IMU传感器快速测试指南

## 🎯 测试目的

验证IMU（惯性测量单元）能否正常工作，并输出：
- **加速度** (ax, ay, az) - 来自加速度计

### **方式3：使用ROS 2命令**

```bash
# 查看完整IMU数据
ros2 topic echo /rexrov/imu

# 只看加速度
ros2 topic echo /rexrov/imu --field linear_acceleration

# 只看角速度
ros2 topic echo /rexrov/imu --field angular_velocity
```

## 📊 数据解读

### **加速度数据**

| 轴 | 名称 | 静止时 | 运动时 |
|----|------|--------|--------|
| **X轴** | 前向加速度 | ~0 m/s² | 前进>0, 后退<0 |
| **Y轴** | 侧向加速度 | ~0 m/s² | 左移>0, 右移<0 |
| **Z轴** | 垂向加速度 | ~9.8 m/s² | 重力加速度 |

**注意**：静止时Z轴约为9.8 m/s²（重力）！

---

### **角速度数据**

| 轴 | 名称 | 静止时 | 旋转时 |
|----|------|--------|--------|
| **X轴** | 横滚角速度 | ~0 rad/s | 左右倾斜 |
| **Y轴** | 俯仰角速度 | ~0 rad/s | 前后倾斜 |
| **Z轴** | 偏航角速度 | ~0 rad/s | 左右旋转 |

## ✅ 正常工作的标志

### **静止时**：
```
加速度: X≈0, Y≈0, Z≈9.8  (Z轴有重力)
角速度: X≈0, Y≈0, Z≈0  (无旋转)
```

### **运动时**：
```
加速度: 数值会变化  (反映运动状态)
角速度: 数值会变化  (反映旋转状态)
```

### **数据质量**：
```
标准差:
  加速度 < 0.1 m/s²
  角速度 < 0.01 rad/s
```

## 🔍 常见问题

### **Q1: IMU数据不变**

**检查**：
```bash
# 检查话题是否发布
ros2 topic hz /rexrov/imu
```

**应该显示**：
```
average rate: 50.0  (50Hz)
```

---

### **Q2: Z轴加速度不是9.8**

**可能原因**：
- 机器人在运动
- 重力补偿已启用

**正常范围**：9.5 - 10.0 m/s²

---

### **Q3: 角速度一直不为0**

**可能原因**：
- 机器人在旋转
- 陀螺仪漂移

**正常范围**：±0.001 rad/s（静止时）
