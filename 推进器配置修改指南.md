## 🔧 修改推进器配置

### 方案1：创建自定义配置文件 ⭐ 推荐

**步骤1：复制默认配置**
```bash
cd ~/ros2_ws/src/uuv_descriptions/robots/
cp rexrov_default.xacro rexrov_custom_6thrusters.xacro
```

**步骤2：修改推进器定义**

编辑 `rexrov_custom_6thrusters.xacro`：

```xml
<?xml version="1.0"?>
<robot xmlns:xacro="http://www.ros.org/wiki/xacro">

  <xacro:property name="prop_mesh_file" value="file://$(find uuv_descriptions)/meshes/prop.dae"/>

  <!-- 推进器宏定义 -->
  <xacro:macro name="thruster_macro"
    params="namespace thruster_id *origin">
    <xacro:thruster_module_first_order_basic_fcn_macro
      namespace="${namespace}"
      thruster_id="${thruster_id}"
      mesh_filename="${prop_mesh_file}"
      dyn_time_constant="0.05"
      rotor_constant="0.00031">
      <xacro:insert_block name="origin"/>
    </xacro:thruster_module_first_order_basic_fcn_macro>
  </xacro:macro>

  <!-- 6推进器配置（去掉T2和T3） -->
  <xacro:thruster_macro namespace="${namespace}" thruster_id="0">
    <origin xyz="-0.890895 0.334385 0.528822" rpy="${0*d2r} ${-74.53*d2r} ${-53.21*d2r}"/>
  </xacro:thruster_macro>

  <xacro:thruster_macro namespace="${namespace}" thruster_id="1">
    <origin xyz="-0.890895 -0.334385 0.528822" rpy="${0*d2r} ${-74.53*d2r} ${53.21*d2r}"/>
  </xacro:thruster_macro>

  <!-- 去掉T2和T3 -->

  <xacro:thruster_macro namespace="${namespace}" thruster_id="4">
    <origin xyz="-0.412125 0.505415 0.129" rpy="${0*d2r} ${0*d2r} ${45*d2r}"/>
  </xacro:thruster_macro>

  <xacro:thruster_macro namespace="${namespace}" thruster_id="5">
    <origin xyz="-0.412125 -0.505415 0.129" rpy="${0*d2r} ${0*d2r} ${-45*d2r}"/>
  </xacro:thruster_macro>

  <xacro:thruster_macro namespace="${namespace}" thruster_id="6">
    <origin xyz="0.412125 0.505415 0.129" rpy="${0*d2r} ${0*d2r} ${135*d2r}"/>
  </xacro:thruster_macro>

  <xacro:thruster_macro namespace="${namespace}" thruster_id="7">
    <origin xyz="0.412125 -0.505415 0.129" rpy="${0*d2r} ${0*d2r} ${-135*d2r}"/>
  </xacro:thruster_macro>

</robot>
```

## 🎯 常见修改场景

### 场景1：4推进器AUV（简化版）

**配置：**
- 保留T0, T1（前部推进器）
- 保留T6, T7（底部推进器，用于垂直和偏航）

**TAM矩阵：**
```python
# 6自由度 × 4推进器
TAM = [
    [0.16, 0.16, -0.71, -0.71],  # X
    [-0.21, 0.21, 0.71, -0.71],  # Y
    [0.96, 0.96, 0.0, 0.0],      # Z
    [0.44, -0.44, -0.09, 0.09],  # Roll
    [0.94, 0.94, 0.09, 0.09],    # Pitch
    [0.14, -0.14, 0.65, -0.65]   # Yaw
]
```

**性能影响：**
- ✅ 前进能力：↓25%（失去T4,T5）
- ✅ 垂直能力：不变（T6,T7仍在）
- ⚠️ 偏航能力：↓50%（失去T4,T5的力矩）

## ⚠️ 注意事项

### 1. TAM矩阵必须匹配
**错误示例：**
```python
# URDF中定义了6个推进器
# 但TAM矩阵仍然是8列
# ❌ 会导致启动失败
```

**正确做法：**
```python
# 修改TAM矩阵，确保列数=推进器数量
tam.shape = (6, 6)  # 6自由度 × 6推进器
```

### 2. 推进器ID必须连续
**错误：**
```xml
<thruster_id="0"/>
<thruster_id="1"/>
<thruster_id="4"/>  <!-- ❌ 跳过2,3 -->
```

**正确：**
```xml
<thruster_id="0"/>
<thruster_id="1"/>
<thruster_id="2"/>  <!-- ✅ 连续 -->
```

### 3. 位置参数要合理
```xml
<!-- xyz: 相对于base_link的位置（米） -->
<!-- rpy: 姿态角（度） -->
<origin xyz="x y z" rpy="roll pitch yaw"/>

<!-- 例如：前方1米，左方0.5米，上方0.3米 -->
<origin xyz="1.0 0.5 0.3" rpy="0 0 0"/>
```
