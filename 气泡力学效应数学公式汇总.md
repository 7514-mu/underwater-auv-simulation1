# 气泡力学效应数学公式汇总

**日期**: 2026-05-16
**版本**: v2.0

---

## 一、阻力增加

### 公式
```
阻力公式：F_drag = 0.5 × ρ × Cd × A_eff × v²

气泡影响（Clift经验公式）：
  A_eff = A_base × (1 + K_AREA × α)

体积分数计算：
  α = bubble_density × α_max
```

### 参数说明

| 参数 | 值 | 单位 | 来源 |
|------|-----|------|------|
| ρ (水密度) | 998.2 | kg/m³ | 标准值 @20°C |
| Cd (阻力系数) | 1.0 | - | 方头ROV |
| A_base (基础截面积) | 0.3 | m² | RexROV 几何参数 |
| K_AREA (Clift经验系数) | 2.5 | - | Clift et al. (1978) 经验范围 2.0~3.5 |
| α_max (体积分数上限) | 0.15 | - | 尾流气泡群典型上限 |

### 理论来源
- **主要依据**：Clift et al. (1978)《Bubbles, Drops, and Particles》多相流阻力公式
- **定性支持**：鲍文春论文指出"气泡覆盖航行体表面，改变了航行体表面压力分布规律"

## 二、浮力变化

### 公式
```
浮力公式：ΔF_buoyancy = ρ_water × g × V_bubble

气泡体积计算：
  V_bubble = V_robot × α

体积分数计算：
  α = bubble_density × α_max
```

### 参数说明

| 参数 | 值 | 单位 | 来源 |
|------|-----|------|------|
| ρ_water (水密度) | 998.2 | kg/m³ | 标准值 @20°C |
| g (重力加速度) | 9.81 | m/s² | 标准值 |
| V_robot (机器人体积) | 0.5 | m³ | RexROV 参数 |
| α_max (体积分数上限) | 0.15 | - | 尾流气泡群典型上限 |

### 计算示例（bubble_density = 0.3）
```
α = 0.3 × 0.15 = 0.045
V_bubble = 0.5 × 0.045 = 0.0225 m³
ΔF_buoyancy = 998.2 × 9.81 × 0.0225 ≈ 220.5 N
```

### 理论来源
- **主要依据**：Archimedes原理（物体在流体中受到的浮力等于排开流体的重量）
- **气泡占据体积但不贡献重量**（ρ_air ≈ 1.2 kg/m³ ≪ ρ_water），净浮力增大

## 三、漂移力

### 公式
```
漂移力公式：F_drift = C_DRIFT × α × v_bubble × μ × A_cross

体积分数计算：
  α = bubble_density × α_max
```

### 参数说明

| 参数 | 值 | 单位 | 来源 |
|------|-----|------|------|
| C_DRIFT (群体效应系数) | 120 | - | Stokes公式推导 + 群体效应修正 |
| v_bubble (气泡上升速度) | 0.3 | m/s | Clift et al. (1978) mm级气泡终端速度 |
| μ (动力粘度) | 1.002×10⁻³ | Pa·s | 20°C标准值 |
| A_cross (横截面积) | 0.3 | m² | RexROV 参数 |
| α_max (体积分数上限) | 0.15 | - | 尾流气泡群典型上限 |

### 气泡终端速度来源
- Clift et al. (1978): mm 级气泡终端速度 0.2~0.35 m/s
- Mendelson (1967) 关联式: v_t = sqrt(2σ/(ρd) + gd/2)
- 对于 R = 1~5mm 气泡，终端速度约 0.2~0.3 m/s

### 计算示例（bubble_density = 0.3）
```
α = 0.3 × 0.15 = 0.045
F_drift = 120.0 × 0.045 × 0.3 × 1.002e-3 × 0.3
        ≈ 0.00049 N ≈ 0.49 mN
```

### 理论来源
- **主要依据**：Stokes粘性阻力公式 (Landau & Lifshitz, 1987)
- 上升气泡带动周围水体 → 对 ROV 产生向上粘性拖曳力

## 四、随机扰动

### 公式
```
扰动力公式：F_perturb = k × α × ξ

其中：
  k  = 40 N (扰动强度系数)
  α  = bubble_density × 0.15
  ξ  = 随机方向向量，偏向上方
```

### 扰动方向特性
```
ξ_x = random(-1, 1) × 0.3   (水平)
ξ_y = random(-1, 1) × 0.3   (水平)
ξ_z = random(0.3, 1)        (偏向上方，浮力方向)
```

### 计算示例（bubble_density = 0.3）
```
α = 0.045
intensity = 40 × 0.045 = 1.8 N
F_perturb_z = 1.8 × random(0.3, 1.0) ≈ 0.5~1.8 N (向上)
F_perturb_x = 1.8 × 0.3 × random(-1, 1) ≈ ±0.54 N (水平)
```

### 理论来源
- 气泡碰撞/破裂力脉冲的工程简化模型
- 方向偏向上方（浮力方向），强度与体积分数成正比

## 五、总合力

### 公式
```
统一体积分数：α = bubble_density × 0.15

F_total = F_drag + ΔF_buoy + F_drift + F_perturb
```

### 方向特性

| 力类型 | 方向 |
|--------|------|
| 阻力 (F_drag) | 与速度方向相反 |
| 浮力变化 (ΔF_buoy) | 向上（z轴正方向） |
| 漂移力 (F_drift) | 向上（z轴正方向） |
| 随机扰动 (F_perturb) | 随机，偏向上方 |

## 六、关键参数来源汇总

| 参数 | 值 | 来源 |
|------|-----|------|
| Cd = 1.0 | 方头 ROV 阻力系数 | ROV 几何参数 |
| K_AREA = 2.5 | Clift et al. (1978) | 经验范围 2.0~3.5 |
| α_max = 0.15 | 尾流气泡群典型上限 | 实验观测 |
| v_bubble = 0.3 m/s | Clift et al. (1978) | mm 级气泡终端速度 |
| C_DRIFT = 120 | Stokes 公式推导 | 群体效应修正 |
| k = 40 N | 工程估算 | 气泡碰撞动量转移 |

## 七、公式推断法说明

本仿真采用**公式推断法**实现：

1. **核心思想**：从经典力学公式出发（Morison方程、Archimedes原理、Stokes阻力），用工程简化参数实现实时计算
2. **优势**：计算效率高，适合 ROS2 + Gazebo 实时仿真
3. **适用场景**：低体积分数气泡群（α < 0.15），mm 级自然/尾流气泡
4. **局限**：不求解 NS 方程或 R-P 方程，气泡间相互作用未建模
